Duplicate Site

As part of a national action against a fashion brand, I created a duplicate site of SHEIN. Using the inspect tool I was able to investigate the layout of the original website further in order to understand the original code and create my own code to produce a replica website. Given my 48 hour deadline I used simple HTML and CSS to create this. As I knew the site would be heavily accessed through QR codes, responsitivity was a key objective of mine, ensuring it was accessible through mobile devices, as well as through desktops.
